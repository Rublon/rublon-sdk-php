Passwordless Login by Rublon

Passwordless Login to sposób integracji systemu uwierzytelniającego Rublon, który umożliwia bezhasłowe logowanie telefonem. Użytkownicy usługi, w której został wdrożony Rublon, mogą logować się do swoich kont skanując kod QR, zamiast wypełniania formularza logowania z hasłem.

Opis procesu

  1. Wygenerowanie przycisku typu small lub large
  2. Przekierowanie przeglądarki na pozyskany adres URL (poprzez kliknięcie w przycisk) a następnie oczekiwanie na zeskanowanie kodu przez użytkownika.
  3. Pobranie danych uwierzytelnionego użytkownika (za pomocą tokenu dostępowego otrzymanego zwrotnie od Rublona - Callback).
  4. Sprawdzenie czy użytkownik istnieje w bazie danych systemu na podstawie otrzymanych danych z Rublona (skróty z adresów email użytkownika). W przypadku gdy użytkownik został odnaleziony należy zalogować go do systemu, w przeciwnym wypadku należy wyświetlić odpowiedni komunikat (np. nie odnaleziono użytkownika, dodaj swój uczelniany adres email do Rublona i spróbuj ponownie).

Przykładowa integracja

Poniżej została opisana przykładowa integracja na podstawie powyższego opisu oraz nawiązując do dostarczonego przykładu napisanego w języku PHP z wykorzystaniem Rublon PHP SDK.

Dane dostępowe do API

Aby skorzystać z API Rublona należy ustawić dane dostępowe.

define('SYSTEM_TOKEN', 'SYSTEM_TOKEN_ZE_STAREGO_SYSTEMU_STUDNET');
define('SECRET_KEY', 'SECRET_KEY_ZE_STAREGO_SYSTEMU_STUDNET');

Uwaga: Dane dostępowe token systemowy oraz klucz poufny trzeba pozyskać dodając projekt na http://developers.rublon.com. W celu dodania projektu należy zalogować się za pomocą aplikacji Rublon, która dostępna jest za darmo w sklepach Play Store, App Store i WindowsPhone.

Nagłówki

Aby skorzystać z metod SDK należy dołączyć następujące pliki dostępne w katalogu libs:

// Require libs
//----------------------------------
require_once 'libs/Rublon/RublonLogin.php';
require_once 'libs/Rublon/Rublon2Factor.php';

Krok #1 i #2

Utworzenie instancji klasy RublonLogin oraz RublonGUI, a następnie wyrenderowanie przycisku typu small

PHP I HTML 

// Create Rublon instance
$rublon = new RublonLogin(SYSTEM_TOKEN, SECRET_KEY);

// Prepare login URL
$loginURL = 'http://' . $_SERVER['HTTP_HOST'] . parse_url($_SERVER['REQUEST_URI'], PHP_URL_PATH) . '?action=rublon';

//Create Rublon GUI instance
$gui = new RublonGUI($rublon);

// Attach JS script
echo $gui->getConsumerScript();

// Rublon Login Box Small
echo new RublonLoginBox($loginUrl);

Po kliknięciu w przycisk następuje przekierowanie na $loginUrl, w której to akcji pobierany jest URL, na który należy przekierować przeglądarkę w celu rozpoczęcia autoryzacji użytkownika.

PHP 

<?php        

// Create Rublon instance
$rublon = new RublonLogin(SYSTEM_TOKEN, SECRET_KEY);

// Prepare CallbackURL
$callbackUrl = 'http://' . $_SERVER['HTTP_HOST'] . parse_url($_SERVER['REQUEST_URI'], PHP_URL_PATH) . '?rublon=callback';

// Request for QR code URL
$url = $rublon->getAuthURL($callbackUrl);

// Redirect to Rublon to start authentication process
header('location:' . $url);

Po uzyskaniu URL następuje przekierowanie w celu uwierzytelnienia użytkownika za pomocą skanowania kodu QR.

Krok #3 - Callback

Gdy użytkownik zeskanuje kod QR i zostanie uwierzytelniony przez Rublon to następuje wywołanie adresu zwrotnego CallbackURL wraz z parametrem token. Token umożliwia jednorazowe pobranie danych uwierzytelnionego użytkownika.

PHP

// Create Rublon instance
$rublon = new RublonLogin(SYSTEM_TOKEN, SECRET_KEY);

// Request for user credentials
$response = $rublon->getCredentials($_GET['token']);

// Get user data array from the response
$userData = $response->getResponse();

PHP

Kod PHP dla przypadków opisanych powyżej:

Tablica symulująca bazę użytkowników systemu 

// List of local users:
$users = array('user@example.com', 'user2@example.com', 'unknown@example.com');

// Create a helper hash-to-email map:
$usersEmailsHashList = array();
foreach ($users as $email) {
    $usersEmailsHashList[hash('sha256', $email)] = $email;
}

Wyszukanie użytkownika w bazie a następnie zalogowanie go do systemu:

if /* requesting Rublon callback */ (!empty($_GET['rublon']) AND $_GET['rublon'] == 'callback') {
    try {
        $rublon = new RublonLogin(SYSTEM_TOKEN, SECRET_KEY); // 
        $credentials = $rublon->getCredentials($_GET['token']);

        if ($userEmail = $credentials->getUserEmail()) {
            if (in_array($userEmail, $users)) {
                $_SESSION['user'] = $userEmail;
            }
        } else {
            $hashList = $credentials->getUserEmailHashList();
            foreach ($hashList as $record) {
                if (isset($usersEmailsHashList[$record['hash']])) {
                    $_SESSION['user'] = $usersEmailsHashList[$record['hash']];
                }
            }
        }

        if (empty($_SESSION['user'])) {
            die('User not found');
        } else {
            header('Location: ./');
            exit;
        }

    } catch (RublonException $e) {
        echo '<pre>';
        echo $e->getClient()->getRawRequest() . PHP_EOL;
        echo $e->getClient()->getRawResponse() . PHP_EOL;
        echo get_class($e) . PHP_EOL;
        die($e->getMessage());
    }
}

Gdy użytkownik zostanie odnaleziony w bazie użytkowników to należy go zalogować do systemu i przekierować na stronę startową dostępną po zalogowaniu.

Osadzenie Menadżera Urządzeń Zaufanych

Rublon umożliwia utworzenie Urządzenia Zaufanego dla każdego uwierzytelnionego użytkownika. Urządzenie Zaufane pozwala na ponowne zalogowanie do danego systemy bez konieczności skanowania kodu QR. W takim przypadku użytkownik uwierzytelniany jest za pomocą Urządzenia Zaufanego. Poniżej przedstawiony jest kod do osadzenia Menedżera Urządzeń Zaufanych przypisanych do danego użytkownika.

PHP

echo '<style type="text/css">iframe {width: 500px; height: 400px; display: inline-block !important; border: 1px solid #cccccc;}</style>';

// Create RublonLogin instance
$rublon = new RublonLogin(SYSTEM_TOKEN, SECRET_KEY);

// Create RublonGUI instance 
$gui = new RublonGUI($rublon, $userId = $_SESSION['user'], $userEmail = $_SESSION['user']);

// Attach RublonConsumerJS
echo $gui->getConsumerScript();

// Render Trusted Device Widget
echo new RublonDeviceWidget();
